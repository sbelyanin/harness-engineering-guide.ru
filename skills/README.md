# Skills

Переиспользуемые skills для workflow Harness Engineering. Каждый skill — самодостаточный пакет с `SKILL.md` и опциональными скриптами.

## Доступные Skills

| Skill | Тип | Описание |
|-------|-----|----------|
| [abuse-hunter](./abuse-hunter/) | перевод | Обнаружение batch-регистраций и злоупотреблений в SaaS — 6-шаговый процесс расследования + 12-балльная оценка + SQL-шаблоны + офлайн-анализ на Python |
| [ru-doc-summarizer](./ru-doc-summarizer/) | RU-оригинал | Саммаризация длинных русскоязычных документов (ГОСТы, регламенты, протоколы) — 5-шаговый процесс, извлечение решений/action items с метками модальности, разрешение ссылок, chunker на Python |
| [152fz-audit](./152fz-audit/) | RU-оригинал | Аудит chat-логов AI-агента на наличие ПДн по 152-ФЗ — 6-шаговый процесс, классификация по УЗ-1..УЗ-4, regex + pseudonymization на Python (email, телефон, ИНН, СНИЛС, паспорт, PAN с Luhn-check) |

## Как использовать

1. Скопируйте папку skill в директорию `skills/` вашего agent workspace
2. Агент автоматически обнаружит и использует skill при релевантных задачах
3. Или обращайтесь к `SKILL.md` напрямую для ручных workflow анализа

## Типы skills

- **перевод** — переведены из upstream-репозитория [nexu-io/harness-engineering-guide](https://github.com/nexu-io/harness-engineering-guide)
- **RU-оригинал** — созданы в рамках русскоязычного издания, ориентированы на задачи и реалии русскоязычной аудитории
