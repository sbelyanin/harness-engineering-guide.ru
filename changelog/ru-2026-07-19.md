# 2026-07-19 — Запуск самостоятельной дорожки RU-издания

Эта запись открывает отдельный changelog русскоязычного издания. Он ведётся параллельно с переводом upstream-обновлений из [nexu-io/harness-engineering-guide](https://github.com/nexu-io/harness-engineering-guide) — собственные изменения RU-издания помечаются префиксом `ru-` в имени файла.

## Track A — Аудит и вычитка

- **A1–A3.** Проверены качество переводов, внутренние ссылки и сверка заголовков `frontmatter ↔ H1 ↔ guide-data.ts`. Английский в прозе отсутствует (только в коде, JSON и намеренно цитируемых промптах — по конвенции).
- **A4.** Выровнен frontmatter во всех **26 статьях** `guide/`: добавлены `title` и `section`. Единый стиль, надёжный fallback без опоры на body H1.
- **A5.** В статью [Scheduling и автоматизация](/guide/scheduling-and-automation) добавлен недостающий body H1 — приведена к единообразию с остальными 25 статьями.

## Track B — Новые RU-оригинальные статьи

- **B1.** Опубликована статья [On-Prem Harness: запуск в air-gapped среде](/guide/on-prem-harness) — локальный inference, vending tools, подписанные bundle’ы, аудит под enterprise-требованиями. Первая статья B-трека после [Russian LLM в Harness](/guide/russian-llm-harness).

## Track C — Skills

- **C1.** Добавлен RU-оригинальный skill [`ru-doc-summarizer`](https://github.com/nexu-io/harness-engineering-guide/tree/main/skills/ru-doc-summarizer) — саммаризация длинных русскоязычных документов (ГОСТы, регламенты, протоколы): 5-шаговый процесс, метки модальности (MUST/SHOULD/MAY), разрешение ссылок, chunker на Python.

## Track D — Сообщество и навигация

- **D2.** Появилась страница [Сообщество](/community) с инструкциями для контрибьюторов и каналами связи.
- **D1.** Запущен этот отдельный changelog с префиксом `ru-` для собственных изменений RU-издания.
